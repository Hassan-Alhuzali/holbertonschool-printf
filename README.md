# printf

## Description
Write your own printf function.

## Files
| File | Description |
|------|-------------|
| main.h | Header file |
| printf.c | Main function |
| functions.c | Helper functions |
| man_3_printf | Man page |

## Compilation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c

## Usage
_printf("Hello %s\n", "World");

## Specifiers
| Specifier | Output |
|-----------|--------|
| %c | Character |
| %s | String |
| %% | Percent sign |
| %d | Integer |
| %i | Integer |
